# 📘 SQL - More Queries

## 📁 Project Info

- **GitHub Repository**: `holbertonschool-higher_level_programming`
- **Directory**: `SQL_more_queries`

---

## 🎯 Project Objective

This project focuses on advanced SQL features, including:

- MySQL user creation and privilege management
- Table constraints: `NOT NULL`, `UNIQUE`, `PRIMARY KEY`, `FOREIGN KEY`
- Multi-table queries using `JOIN`, subqueries, and `UNION`
- Database normalization principles and relational schema design

---

## 🧠 Learning Objectives

By the end of this project, you should be able to:

- Create new MySQL users and assign privileges
- Understand and apply constraints like `PRIMARY KEY`, `FOREIGN KEY`, `NOT NULL`, and `UNIQUE`
- Retrieve data from multiple tables using `JOIN`, subqueries, and unions
- Understand how to manage normalized relational data models
- Use MySQL 8.0 features and best practices for SQL style

---
